UI Developer Co-op at Distill Mobile

Distill Mobile is seeking an Intermediate Web UI Developer (co-op) to work on client applications.

RESPONSIBILITIES:

  • Contributing to existing mobile and web e-commerce applications developed for clients
  • Implementing web user interfaces using various technologies including but not limited to JavaScript, Ember, HTML5, JQuery and CSS3
  • Building adaptive page layouts
  • Reviewing, testing, and debugging application code
  • Maintaining and testing web applications
  • Performing other related duties as assigned

QUALIFICATIONS:

  • Enrolled in 3rd or 4th year of a Bachelor’s degree or graduate degree in Computer Science or related discipline
  • Minimum 1 to 2 years of experience in developing web applications
  • Experience in developing cross-browser UIs for web applications using modern frameworks
  • Strong communication skills
  • Experience with Ruby On Rails or Node.js is a bonus
  • Experience with PostgreSQL or MySQL is a bonus
  • Experience with Android or iOS development is a bonus
  • Ability to finish projects on time, within budget and with high quality

Please reply Tom Murphy with your cover letter and resume to be considered for this role.
